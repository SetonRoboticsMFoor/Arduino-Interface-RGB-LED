# Arduino-Interface-RGB-LED

This project includes a basic interface html page that allows the user to connect their Arduino Mega to the web interface and control an RGB LED.

When connecting the RGB LED use a 330 Ohm resistor and connect to pin 2 on the Arduino.
For the green LED connect to a 100 Ohm resistor to pin 3.
For the blue LED connet to a 100 Ohm resistor to pin 4.
